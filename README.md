# 🎓 Campus Connect

An intelligent, AI-powered platform that transforms campus placements and research opportunities into a personalized, data-driven journey for students.

---

## 📌 Problem Statement

The campus placement process is mostly manual and scattered, leaving students unsure of which roles match their skills. Traditional portals act like simple job boards with no personal guidance, and ATS filters often hide resumes that aren’t well-optimized.

Research-focused students also lack support for finding mentors or academic opportunities.

👉 A smart, centralized platform is needed to:
- Guide students effectively  
- Highlight skills to build  
- Connect them to the right career and research paths  

---

## 🎯 Project Objectives

The primary objectives of Campus Connect are:

- 🤖 Provide AI-based job matching aligned with student skills  
- 📄 Automate resume evaluation using ATS-style semantic analysis  
- 📊 Identify skill gaps and guide targeted upskilling  
- 🎓 Support both corporate placements and research internships  
- 😌 Reduce confusion and anxiety during placement seasons  

---

## 🏗️ System Architecture & Tech Stack

Campus Connect follows a modular hybrid architecture for scalability and maintainability.

### 💻 Frontend
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Fetch API (asynchronous data handling)  
- Web Speech API (voice interaction)

### ⚙️ Backend
- Python (Flask)  
- RESTful APIs  

### 🗄️ Data Management
- JSON (placement data)  
- SQLite Database  
- SQLAlchemy ORM  

---

## 🤖 Artificial Intelligence Integration

Campus Connect integrates multiple AI models to enhance decision-making:

| AI Function | Purpose |
|------------|--------|
| Smart Job Matching | Matches student skills with job descriptions |
| ATS Resume Checker | Evaluates resume readiness |
| Job Description Formatter | Converts raw job data into readable format |
| Job Role Explainer | Simplifies complex job roles |
| Skill Extraction Engine | Identifies key required skills |
| Company Insight Generator | Generates company summaries |

👉 These AI components provide meaningful insights instead of static listings.
